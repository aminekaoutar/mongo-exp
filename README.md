# Projet CRUD – Réservation d'hôtel

## Description
Ce projet est une application de **réservation d'hôtel** utilisant **Express** et **MongoDB**.  
Le but est de gérer les chambres et les réservations, avec un panier pour ajouter/supprimer des réservations.

## Fonctionnalités
- **CRUD sur les chambres :** créer, lire, modifier et supprimer des chambres.
- **Gestion du panier / réservation :** ajouter et supprimer des chambres réservées, calcul du total.
- **Backend :** Express
- **Base de données :** MongoDB avec Mongoose
